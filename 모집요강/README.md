# 모집요강

## Requirement
- Flutter = 3.3.10
- Dart SDK = 2.18.6
<br><br>
- table_calendar = 3.0.9
- dropdown_button2 = 2.1.0
- circular_chart_flutter: ^0.0.2

<br>

## 구현된 기능

### 1. 모집요강
- 학교 홍보 : None
- 주요 변경 사항 :
    - 입학처 모집요강의 정보 copy
- 모집 인원
    - 대학 & 학과 검색창
    - 학과별 모집 인원 테이블
- 전형 일정
    - Custom 달력

### 2. 전형가이드
- 검색창
- 검색한 전형의 설명
    - 평가 비율의 원형 그래프
    - 지원할 수 있는 학과에 대한 테이블
    - 전형비
    - 제출 문서
    - 세부 사항
